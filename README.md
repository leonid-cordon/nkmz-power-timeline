# NKMZ Power Timeline

Веб-страница для наглядной визуализации отключений электропитания на серверной площадке НКМЗ по данным системного журнала Windows (SRV3).

## Что показывает дэшборд

Данные берутся из журнала событий Windows (события **6005** и **6008**) на сервере **SRV3** и заранее агрегируются в JSON-файл.

На странице можно увидеть:

- статистику по **годам**:  
  количество аварий, количество дней с отключениями, суммарное время без питания;
- сводку по **месяцам**:  
  для каждого месяца — сколько было аварий и сколько часов не было напряжения;
- **календарь выбранного месяца**:  
  дни с отключениями подсвечиваются, внутри каждой ячейки есть мини-шкала 0–24 часов с отметками интервалов без питания;
- **детализацию дня**:  
  горизонтальная шкала 0–24 часов, на которой красным выделены интервалы, когда не было электропитания.

## Структура файлов

В корне репозитория находятся:

- `nkmz_power_outages.html` — основная HTML-страница;
- `nkmz_power_outages.js` — клиентская логика (загрузка JSON, отрисовка годов, месяцев, календаря и шкалы по суткам);
- `power_stats_all_years.json` — агрегированные данные по отключениям за все годы (2015–2025), сформированные из журнала событий Windows по событиям 6005/6008.

## Как открыть через GitHub Pages

1. Убедитесь, что в корне репозитория лежат три файла:
   - `nkmz_power_outages.html`
   - `nkmz_power_outages.js`
   - `power_stats_all_years.json`

2. В настройках репозитория откройте **Settings → Pages**:
   - в разделе **Source** выберите: `Deploy from a branch`;
   - **Branch**: `main` и папка `/ (root)`;
   - сохраните настройки.
